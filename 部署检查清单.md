## 🚀 快速部署命令

### Nginx

```bash
# 上传文件到服务器
scp -r . user@server:/var/www/hyx-website

# 配置 Nginx
sudo cp nginx.conf /etc/nginx/sites-available/hyx-website
sudo ln -s /etc/nginx/sites-available/hyx-website /etc/nginx/sites-enabled/

# 测试并重启
sudo nginx -t
sudo systemctl restart nginx

# 配置 HTTPS
sudo certbot --nginx -d your-domain.com
```

### Docker

```bash
# 构建镜像
docker build -t hyx-website .

# 运行容器
docker run -d -p 80:80 --name hyx-site hyx-website
```

### 阿里云 OSS

```bash
# 使用 ossutil 上传
ossutil cp -r . oss://your-bucket/

# 启用静态网站托管
# 在 OSS 控制台配置
```

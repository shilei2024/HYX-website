# 弘易芯科技官网 - 部署文档

## 📖 文档说明

本文档提供从零开始的完整部署指南，适用于 Docker Compose v2.23.0+（推荐 v5.0+）。

---

## 🎯 快速开始

### 方式一：自动部署（推荐）

```bash
# 1. 登录服务器
ssh user@your-server-ip

# 2. 进入网站目录
cd /path/to/HYX-website

# 3. 设置权限
chmod +x setup-permissions.sh && ./setup-permissions.sh

# 4. 执行一键部署
./一键部署.sh
```

### 方式二：手动部署

详见 **[部署指南.md](部署指南.md)**

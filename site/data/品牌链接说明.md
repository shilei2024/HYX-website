# 品牌官网链接修改说明

各品牌卡片上的「访问官网」链接来自以下 JSON 文件中的 **`url`** 字段，直接修改对应文件即可生效。

## 一、代理品牌（产品中心 → 代理品牌）

**文件路径：** `data/brands.json`

每个品牌是一个对象，修改其中的 `"url": "https://..."` 即可：

```json
{
  "id": 1,
  "name": "华朔",
  "logo": "assets/brands/华朔.png",
  "url": "https://这里改成该品牌官网地址/",
  "description": "中低压 N & P MOSFETs。"
}
```

- 链接需带 `https://` 或 `http://`
- 若暂不展示官网链接，可改为 `"url": "#"`

## 二、分销品牌（产品中心 → 分销品牌）

**文件路径：** `data/distribution-brands.json`

同样只改每个品牌里的 **`url`** 字段即可，例如：

```json
{
  "name": "兆易创新",
  "url": "https://www.gigadevice.com/",
  ...
}
```

修改后保存文件，刷新对应页面即可看到新链接（点击「访问官网」会新窗口打开）。

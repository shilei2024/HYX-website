# 代理品牌 Logo 放置说明

## 提高 Logo 清晰度（重要）

为避免放大后模糊，请按以下优先级准备 Logo 文件：

1. **首选 SVG 矢量格式**（最清晰，任意放大不失真）：如 `华朔.svg`
2. **高分辨率 PNG**：建议尺寸 **400×200px 以上**，背景透明
3. **避免使用小图放大**：原始图小于 200px 宽会模糊

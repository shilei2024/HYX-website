## 🛠️ 部署脚本

| 脚本 | 用途 |
|------|------|
| `一键部署.sh` | 一键完成所有部署步骤 |
| `manage-watch.sh` | Watch 模式管理（启动/停止/查看） |
| `install-service.sh` | 安装 systemd 服务（开机自启） |
| `setup-permissions.sh` | 设置文件权限 |

## 📋 使用流程

### 首次部署

1. 阅读 **[README.md](README.md)** 了解项目
2. 按照 **[部署指南.md](部署指南.md)** 进行部署
3. 部署完成后，网站即可访问

### 日常更新

1. 在本地修改代码并提交到 Git
2. 在服务器上执行 `git pull`
3. 网站自动更新（3-5 秒）
4. 详见 **[日常更新.md](日常更新.md)**

### 遇到问题

1. 查看 **[故障排查.md](故障排查.md)**
2. 按照文档排查问题
3. 如无法解决，联系技术支持

## 🎯 快速命令参考

```bash
# 部署
./一键部署.sh

# 更新
git pull

# Watch 管理
./manage-watch.sh status    # 查看状态
./manage-watch.sh start-bg  # 后台启动
./manage-watch.sh stop      # 停止

# 容器管理
docker compose ps           # 查看状态
docker compose logs -f      # 查看日志
docker compose restart      # 重启

# systemd 服务（如已安装）
sudo systemctl status hyx-watch
sudo systemctl restart hyx-watch
sudo journalctl -u hyx-watch -f
```

# 意见反馈表单配置说明

本站「联系我们」页的意见反馈采用 **Formspree** 无感提交：客户填写后点击提交，数据会发送到 Formspree，由 Formspree 将内容转发到您指定的邮箱，同时页面跳转到「提交成功」页，客户无需使用自己的邮箱。

## 配置步骤（约 2 分钟）

1. 打开 [https://formspree.io](https://formspree.io)，使用邮箱免费注册并登录。
2. 点击 **New Form** 创建一个新表单，表单名称可填「弘易芯官网反馈」。
3. 创建成功后，在表单详情页可以看到 **Form Endpoint**，形如：  
   `https://formspree.io/f/xxxxxxxx`  
   其中 `xxxxxxxx` 即为表单 ID。
4. 在项目根目录打开 `data/config.json`，找到 `contact` 节点，将 `formspreeFormId` 填为上述 ID（只填 ID 部分即可，不要带 `https://formspree.io/f/`）：
   ```json
   "contact": {
     "formspreeFormId": "xxxxxxxx",
     "phone": "13823674897",
     ...
   }
   ```
5. 在 Formspree 后台可设置接收邮件的地址（默认是注册邮箱），也可开启邮件通知、查看提交记录等。

配置完成后，客户在「联系我们」页提交表单时会无感提交，成功后自动跳转到「提交成功」页面；您会在邮箱中收到包含姓名、邮箱、主题、留言的邮件。

若未配置 `formspreeFormId`，提交时页面会提示先完成上述配置。
